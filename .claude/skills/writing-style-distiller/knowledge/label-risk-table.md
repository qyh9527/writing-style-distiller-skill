# 高风险标签行为化转换对照表

将文学分析、角色设定中的「抽象情绪/性格标签」转换为「可观察的文本行为/句法规则」。

## 转换原则

- 保留标签仅作为紧凑的标题索引。
- 可执行部分必须是文本中可观测的行为：对话、动作、句法、段落、感官聚焦、叙述距离。
- 若目标模型是 Gemini，进一步减少标签，增加行为列表和示例边界。
- **Gemini 标签连锁退化警告**：Gemini 拆解标签时倾向极端化——将一个多样、因人而异的标签拆成一串单一、高频、八股的子行为。解决路径优先级：①规避性格标签本身 → ②直接描述具体行为（代价：占用大量 token）→ ③用示例片段代替标签定义。
