# 高风险标签行为化转换对照表

将文学分析、角色设定中的「抽象情绪/性格标签」转换为「可观察的文本行为/句法规则」。

## 转换原则

- 保留标签仅作为紧凑的标题索引。
- 可执行部分必须是文本中可观测的行为：对话、动作、句法、段落、感官聚焦、叙述距离。
- 若目标模型是 Gemini，进一步减少标签，增加行为列表和示例边界。

## 转换对照表

| 原始标签 | 常见模型失败 | 行为化替换规则 |
|---|---|---|
| 羞耻 / shame | 脸红崩溃循环、口吃、失去自主性 | 延迟回复、话题转移、截断语法、自我保护姿态（抓紧衣带、移开视线）、控制性沉默、边界主张 |
| 破碎 / broken | 剧情性崩溃、无助感、melodramatic collapse | 碎片化段落节奏（三五字短句）、省略解释、压力下的实际行动（在痛苦中安静做日常事务）、抹去情绪性副词 |
| 疯感 / madness | 随机残酷、语无伦次、狂笑 | 有限度的破规则语法、强迫性母题复现、不稳定注意力转移（关注场景外微不足道的东西）、仅在极端对话时让语法破格 |
| 压抑 / repression | 抽象悲伤、沉重叙述、无病呻吟 | 缩窄感官视野（只盯方寸物品）、日常动作、未说出的对话、受限动词、叙述距离拉远 |
| 温柔 / warmth | 泛化治愈、强行甜蜜、工业糖精 | 具体关怀动作（记住对方食物过敏并默默挪开盘子）、柔化节奏、细心的物品记忆、低压力对话 |
| 活人感 / liveliness | 随机搞笑、现代梗泛滥、吐槽役八股 | 贴合语境的打断、不完整回答、小矛盾（言行不一但符合常理）、实际生存优先级 |
| 傲娇 / tsundere | 单调"哼，才不是" | 否认时语法变短+语速加快、独处时放松、条件性关怀行为（通过非语言妥协暗示） |
| 病娇 / yandere | 失控暴力、刻板瞪眼 | 占有性记忆细节（报出对方微调衣饰、眼神停顿位置）、平静语调嵌入异常逻辑、过度关注行踪 |
