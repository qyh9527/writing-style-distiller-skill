# 优秀文风提示词结构范式

本文件收录经实测验证的优秀文风提示词结构范式，供 Phase 2 诊断和 Phase 3 起草时作为结构参考。

注意：这些范式是**结构模板**，不是内容模板。学习的是组织方式和维度选择，不是具体规则内容。

## 1. 语言特征类：多层语法分析范式

来源：红楼梦风格案例

**结构**：从语言学四层分析法（语音→词汇→语法→语用）逐层描述，每层有独立的规则体系。

```
适用场景：古风、文言、方言、特定时代白话等语域转换要求高的文风
结构骨架：
  1. 语音层面：节奏韵律、句式长短
  2. 词汇层面：虚词来源、实词风格、时代用词
  3. 语法层面：句式结构、省略习惯、定语/流水句偏好
  4. 语用层面：身份对应的语言分层、情绪与笔法的反向关系
  5. 风格正/反例对：用"错误→正确"对照锁定语域边界
```

**关键启示**：
- 正/反例对是语言特征类文风的必备武器，可以精确锁定AI容易穿帮的语域边界
- 语用层面的"身份对应语言分层"解决了同一风格内不同角色说话方式的问题
- "情绪激烈处反用简笔"这类反直觉规则，对AI的效果特别好

## 2. 叙事结构类：内心独白引擎范式

来源：伏见司式青春喜剧案例

**结构**：以一个「思维模拟核心循环」为引擎，配合铁律禁令和节奏控制。

```
适用场景：第一人称内心独白驱动、吐槽役、意识流等需要模拟思维流程的文风
结构骨架：
  1. 视角锁定与核心基调
  2. 思维模拟循环（感知→过度解读→情感否认→自我修正）
  3. 认知标记词列表（等等、不对、话说回来）
  4. 对话格式硬规则（独立成段、禁引导词、禁插入叙述）
  5. 铁律区（用【禁止XX】标记，与一般规则明确区分）
  6. 节奏控制与标点规范
```

**关键启示**：
- 「思维模拟循环」比静态的"写内心独白"有效得多——它给了模型一个可执行的步骤序列
- 铁律用醒目标签标记（【禁止情绪通胀】【禁止文学化修辞】），与一般偏好清晰区分
- 「最高准则」单独一行置顶，解决了模型在众多规则中忘记核心目标的问题

## 3. 对话驱动类：声口+温度层范式

来源：骚话文风（现代版+古风版）

**结构**：以「声口」定义说话人格，以「技法」定义对话的变化策略，以「忌」定义边界。

```
适用场景：对话密集型文风、角色扮演对话风格、需要对话承载风格辨识度的场景
结构骨架：
  1. 说明：一句话定位（对话驱动、学语感不学内容）
  2. 底色：文风的情感基调和人格保护声明
  3. 声口：说话人的整体口吻和态度
  4. 技法：
     - 用词策略（粗雅混搭的节奏）
     - 称呼升级引擎（一场内称呼递进不回头）
     - 温度层切换（央求→放浪→调戏→假装抗拒→...）
     - 姿态不锁定（强势也会软下来，黏糊也会按住人）
     - 花样翻新要求（每种调子唱久了必须换）
  5. 忌：禁止事项
  6. 示例：多段覆盖不同温度层和姿态的示例
```

**关键启示**：
- "学语感学技法不学具体内容"——这句话完美解决了示例被当作剧情素材复用的问题
- 「底色」段明确声明"骚话不改变人格"，防止角色在对话中人格滑坡
- 古风版增加了"粗雅混搭"维度，展示了同一范式在不同语域下如何变体

## 4. 场景响应类：状态机+机制互动范式

来源：NSFW 现实化指导

**结构**：定义多个独立机制，每个机制有触发条件和行为响应，机制之间有交互规则。

```
适用场景：需要根据场景状态动态切换写法的复杂系统（NSFW、战斗、情绪递进等）
结构骨架：
  1. 核心原则（总纲，所有机制的最高约束）
  2. 基础映射区（静态的部位/区域/属性分级表）
  3. 动态机制组（每个机制独立定义）：
     - 触发条件
     - 行为响应
     - 与其他机制的交互（并存/互斥/优先级）
  4. 写作禁忌（独立成段，禁止使用机制术语进入正文）
```

**关键启示**：
- 机制之间的交互规则至关重要（如"身体适应机制的润滑触发不影响疼痛机制的触发，能同时共存触发"）
- "严禁正文出现机制分级术语"——规则是内在逻辑参考，不是可以出现在小说正文中的词汇
- 力度分级表用「无痕→微红→红痕→损伤」这种具体可观测的梯度，比"轻→中→重"精确得多

## 5a. 场景节奏循环范式

来源：男性视觉小说（优秀文风案例4）

**结构**：以场景节奏循环模板作为高层次节奏指导，配合文体定义和少量书写规则。

```
适用场景：文风辨识度主要来自"感觉"和"节奏"的场景，尤其是NSFW/情感交织场景
结构骨架：
  1. 文体定义（2-3句定义核心审美视角）
     - 男性凝视滤镜（静态画面+动作瞬间+官能冲击）
     - 视觉小说（如漫画分镜，口述风格）
  2. 书写规则（3-4条核心规则）
     - 毫不避讳谈论性事与欲望
     - 关注官能描写与性爱感受变化
     - 通过语言/用词/语气刻画角色
     - 对话独立成段省略标签，用动作承接对话
  3. 场景节奏循环模板
     - 身临其境（细腻描绘女性姿态神情衣着）
     → 言笑晏晏（大段对话推动关系升温）
     → 切肤之亲（肉体接触承接对话情绪释放）
     → 循环往复（接触后回归对话或新的静态描写）
```

**关键启示**：
- 节奏循环模板比段落长度规则更高效——它告诉模型"什么时候该写什么"而非"每段多少字"
- 对话靠动作承接而非标签（不写"她说"，写"她放下杯子"然后接对话）
- "男性凝视三要素"（静态+动作+官能）是一种可迁移的审美视角框架
- 规则极少（仅3-4条），靠框架本身传达风格，适合示例不够但节奏感明确的文风

## 5b. 关键词串+感官氛围范式

来源：古风晋江文学（优秀文风案例5）

**结构**：关键词串快速定位模型潜空间，各维度一句话描述，配合严格的比喻分域规则。

```
适用场景：模型训练数据中已有大量相关风格语料的场景（如古风、校园、都市言情）
结构骨架：
  1. 关键词串（一行标签快速对齐AI理解）
     - 中国古风、晋江文学、女性向、粘腻、细腻、暧昧、香艳
  2. 各维度一句话描述
     - 文字风格、香艳描写、形象描写、感官描写、氛围感、留白、情感表达
  3. 比喻规范（精确分域规则）
     - 身体描写：可少量使用比喻与修辞
     - 心理/神情/其他事物：严禁抽象化具象化类比
  4. 示例段落（一段完整的风格展示）
  5. 段落节奏 + 角色语言风格 + 叙事节奏
```

**关键启示**：
- "关键词串"开头法可以快速对齐 AI 的理解——适合模型已有大量相关训练数据的风格
- 比喻分域规则（身体可比喻/心理禁比喻）是精确规则的典范——比"谨慎使用比喻"精确100倍
- 与红楼梦范式走不同路线：红楼靠语法规则精确锁定语域，古风晋江靠感官氛围精确锁定体验
- 框架越简、示例越精准，文风的一致性反而越好——过多规则容易互相打架

## 6. 三层框架公式（新手入门）

来源：知识2 通用文风创作方法论

**公式**：**文风 = 基调 + 文体特征 + 叙述规则**

三者不必俱全，可根据需要组合使用。

```
适用场景：新手首次创作文风、不知道从何下手时的入门脚手架
三层定义：
  1. 基调（核心审美追求）
     - 相当于主提示词，是文风最高层次的概念
     - 如：主观真实论、独白写法、限知视角
     - 本质是告诉模型"这个文风的精神是什么"
  2. 文体特征（文字带给读者的感受）
     - 与文体学相关，关注审美价值与读者体验
     - 如：通俗化、去严肃化、虚拟性、文学性、重情绪、画面感
     - 本质是告诉模型"读者读这种文字应该是什么感受"
  3. 叙述规则（具体的写作技巧）
     - 对话规则、非对话叙述规则、独白规则、表达方式、文字编排、语言体式、禁止项
     - 本质是告诉模型"具体怎么写"
```

**与 6 大家族的映射**：

| 三层框架层级 | 对应家族侧重 |
|---|---|
| 基调层（最抽象） | 情感表达类的情绪底色、叙事结构类的视角锁定 |
| 文体特征层 | 所有家族共用——读者体验是所有文风的目标 |
| 叙述规则层（最具体） | 语言特征类的语法规则、对话驱动类的对话规则、场景响应类的状态机规则 |

**关键启示**：
- 三层框架是入门脚手架，不是终态——熟练后应根据目标家族选择更精确的结构范式
- 新手常见错误是只写基调层（"我想要温暖的文风"），缺少叙述规则层的具体约束
- 三层的递进关系（抽象→中间→具体）本身也是诊断用户需求层级的思路
