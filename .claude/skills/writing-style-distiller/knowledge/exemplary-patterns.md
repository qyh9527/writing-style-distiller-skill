# 优秀文风提示词结构范式

本文件收录经实测验证的优秀文风提示词结构范式，供 Phase 2 诊断和 Phase 3 起草时作为结构参考。

注意：这些范式是**结构模板**，不是内容模板。学习的是组织方式和维度选择，不是具体规则内容。

## 1. 语言特征类：多层语法分析范式

来源：红楼梦风格案例

**结构**：从语言学四层分析法（语音→词汇→语法→语用）逐层描述，每层有独立的规则体系。

```
适用场景：古风、文言、方言、特定时代白话等语域转换要求高的文风
结构骨架：
  1. 语音层面：节奏韵律、句式长短
  2. 词汇层面：虚词来源、实词风格、时代用词
  3. 语法层面：句式结构、省略习惯、定语/流水句偏好
  4. 语用层面：身份对应的语言分层、情绪与笔法的反向关系
  5. 风格正/反例对：用"错误→正确"对照锁定语域边界
```

**关键启示**：
- 正/反例对是语言特征类文风的必备武器，可以精确锁定AI容易穿帮的语域边界
- 语用层面的"身份对应语言分层"解决了同一风格内不同角色说话方式的问题
- "情绪激烈处反用简笔"这类反直觉规则，对AI的效果特别好

## 2. 叙事结构类：内心独白引擎范式

来源：伏见司式青春喜剧案例

**结构**：以一个「思维模拟核心循环」为引擎，配合铁律禁令和节奏控制。

```
适用场景：第一人称内心独白驱动、吐槽役、意识流等需要模拟思维流程的文风
结构骨架：
  1. 视角锁定与核心基调
  2. 思维模拟循环（感知→过度解读→情感否认→自我修正）
  3. 认知标记词列表（等等、不对、话说回来）
  4. 对话格式硬规则（独立成段、禁引导词、禁插入叙述）
  5. 铁律区（用【禁止XX】标记，与一般规则明确区分）
  6. 节奏控制与标点规范
```

**关键启示**：
- 「思维模拟循环」比静态的"写内心独白"有效得多——它给了模型一个可执行的步骤序列
- 铁律用醒目标签标记（【禁止情绪通胀】【禁止文学化修辞】），与一般偏好清晰区分
- 「最高准则」单独一行置顶，解决了模型在众多规则中忘记核心目标的问题

## 3. 对话驱动类：声口+温度层范式

来源：骚话文风（现代版+古风版）

**结构**：以「声口」定义说话人格，以「技法」定义对话的变化策略，以「忌」定义边界。

```
适用场景：对话密集型文风、角色扮演对话风格、需要对话承载风格辨识度的场景
结构骨架：
  1. 说明：一句话定位（对话驱动、学语感不学内容）
  2. 底色：文风的情感基调和人格保护声明
  3. 声口：说话人的整体口吻和态度
  4. 技法：
     - 用词策略（粗雅混搭的节奏）
     - 称呼升级引擎（一场内称呼递进不回头）
     - 温度层切换（央求→放浪→调戏→假装抗拒→...）
     - 姿态不锁定（强势也会软下来，黏糊也会按住人）
     - 花样翻新要求（每种调子唱久了必须换）
  5. 忌：禁止事项
  6. 示例：多段覆盖不同温度层和姿态的示例
```

**关键启示**：
- "学语感学技法不学具体内容"——这句话完美解决了示例被当作剧情素材复用的问题
- 「底色」段明确声明"骚话不改变人格"，防止角色在对话中人格滑坡
- 古风版增加了"粗雅混搭"维度，展示了同一范式在不同语域下如何变体

## 4. 场景响应类：状态机+机制互动范式

来源：NSFW 现实化指导

**结构**：定义多个独立机制，每个机制有触发条件和行为响应，机制之间有交互规则。

```
适用场景：需要根据场景状态动态切换写法的复杂系统（NSFW、战斗、情绪递进等）
结构骨架：
  1. 核心原则（总纲，所有机制的最高约束）
  2. 基础映射区（静态的部位/区域/属性分级表）
  3. 动态机制组（每个机制独立定义）：
     - 触发条件
     - 行为响应
     - 与其他机制的交互（并存/互斥/优先级）
  4. 写作禁忌（独立成段，禁止使用机制术语进入正文）
```

**关键启示**：
- 机制之间的交互规则至关重要（如"身体适应机制的润滑触发不影响疼痛机制的触发，能同时共存触发"）
- "严禁正文出现机制分级术语"——规则是内在逻辑参考，不是可以出现在小说正文中的词汇
- 力度分级表用「无痕→微红→红痕→损伤」这种具体可观测的梯度，比"轻→中→重"精确得多

## 5. 示例驱动+轻量框架范式

来源：男性视觉小说、古风晋江文学

**结构**：极简框架定义核心维度，示例承担主要教学功能。

```
适用场景：文风辨识度主要来自"感觉"而非可列举规则的场景
结构骨架（三段式）：
  方式A（男性视觉小说）：
    1. 文体定义（2-3句定义核心审美视角）
    2. 书写规则（3-4条核心规则）
    3. 场景节奏（身临其境→言笑晏晏→切肤之亲→循环往复）
  
  方式B（古风晋江文学）：
    1. 风格关键词（一行标签串）
    2. 各维度一句话描述（文字风格、描写、氛围、节奏等）
    3. 示例内容
    4. 段落节奏 + 语言风格 + 叙事节奏
```

**关键启示**：
- 场景节奏模板（身临其境→言笑晏晏→切肤之亲→循环往复）是一种高层次的节奏指导，比段落长度规则更有效
- "关键词串"开头可以快速对齐AI的理解，适合模型已有大量相关训练数据的风格
- 框架越简、示例越精准，文风的一致性反而越好——过多规则容易互相打架
