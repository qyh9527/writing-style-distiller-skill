## 1. 人称-聚焦分类体系

基于热奈特（Gérard Genette）叙事学理论，人称与聚焦是两个独立维度：

- **人称**（谁说 / Voice）：叙述者用什么代词讲故事。
- **聚焦**（谁看 / Focalization）：信息通过谁的感知过滤。

### 1.1 三种人称

| 人称 | 代词 | 叙述距离 | 典型体裁 |
|---|---|---|---|
| **第一人称** | 我/我们 | 最近——读者贴着叙述者的意识走 | 自传体、日记体、意识流、轻小说吐槽役、AIRP 角色扮演 |
| **第二人称** | 你 | 强制代入——打破第四面墙，将读者拉进角色 | 互动小说、CYOA、AIRP 对话、实验文学 |
| **第三人称** | 他/她/它/他们 | 可远可近——从贴肩到上帝视角自由调控 | 绝大多数小说、网文、轻小说、严肃文学 |

### 1.2 三种聚焦（热奈特三分法）

| 聚焦类型 | 公式 | 信息范围 | 文风影响 |
|---|---|---|---|
| **零聚焦（全知）** | 叙述者 > 人物 | 无限制：可进入任何角色内心、穿越时空 | 叙述声音统一、可夹叙夹议、段落可自由跳跃场景 |
| **内聚焦（受限）** | 叙述者 = 人物 | 只知锚定角色的所见所感所想 | 文字被角色认知染色、不可描写角色看不见的事、情绪与用词贴合角色个性 |
| **外聚焦（客观）** | 叙述者 < 人物 | 只描述外部行为，不深入任何内心 | 白描为主、海明威式冰山、对话承载几乎所有信息、段落冷峻精简 |

### 1.3 常用组合

| 组合名 | 人称×聚焦 | 说明 | 经典代表 |
|---|---|---|---|
| 第一人称内聚焦 | 我 × 内聚焦 | 最常见的第一人称：只说"我"看到和想到的 | 《麦田里的守望者》、轻小说吐槽役 |
| 第三人称受限 | 他/她 × 内聚焦 | 最流行的现代叙事：贴肩视角 | 《哈利·波特》、绝大多数网文 |
| 第三人称全知 | 他/她 × 零聚焦 | 传统经典叙事：上帝之眼 | 狄更斯、托尔斯泰、汪曾祺部分作品 |
| 第三人称客观 | 他/她 × 外聚焦 | 冷摄影机视角 | 海明威《白象似的群山》、卡佛 |
| 第二人称内聚焦 | 你 × 内聚焦 | AIRP 角色扮演的主流模式 | SillyTavern RP、互动小说 |
| 不可靠叙述者 | 通常第一人称 | 叙述者所说不可全信 | 《洛丽塔》、太宰治《人间失格》 |
| 多重/不定式内聚焦 | 第三人称交替 | 章节间切换聚焦角色 | 《冰与火之歌》、群像叙事 |

## 2. 各视角对文风的具体影响

### 2.1 第一人称 → 文风影响

| 维度 | 影响 |
|---|---|
| **叙述声音** | 必须有独特的说话腔调——口头禅、用词偏好、句式习惯构成风格辨识度 |
| **信息受限** | 不能写"我"不知道的事，不能描写"我"的外貌（除照镜子等自然场景） |
| **内心独白** | 可以自由展开思维流——联想、自我否定、自嘲、犹豫是天然工具 |
| **段落节奏** | 易碎片化：意识跳跃时短句密集、平静时句子舒展 |
| **对话处理** | "我说"可以省略，其他人的对话需要通过"我"的感知过滤 |
| **描写风格** | 所有描写带有"我"的情绪滤镜——同一场景，乐观者和抑郁者描写截然不同 |
| **感官层级** | 第一人称更容易调动感官，因为感知本就是"我"在体验 |

### 2.2 第二人称 → 文风影响

| 维度 | 影响 |
|---|---|
| **代入感** | 最强代入——但也最容易引发读者抗拒（"我才不会这样做"） |
| **叙述声音** | 两种模式：①"你"就是读者自己 ②有个"我"在对"你"说话（如《你一生的故事》） |
| **描写风格** | 倾向指令式或观察式——"你走进房间，闻到咖啡的香气" |
| **段落节奏** | 适合短句推进——祈使语气天然简洁 |
| **AIRP 特殊性** | 角色扮演中第二人称是默认人称，"你"= user 的角色 |
| **风险** | 长篇使用容易疲劳，需要变化节奏来缓解 |

### 2.3 第三人称受限 → 文风影响

| 维度 | 影响 |
|---|---|
| **叙述距离可调** | 核心优势：可以在"贴肩"和"拉远"之间自由切换 |
| **贴肩时** | 像第一人称：自由间接引语（不用引号直接写角色内心）、用词被角色认知染色 |
| **拉远时** | 像摄影机：客观描述动作和场景、压缩时间、交代背景 |
| **对话处理** | 可以写"他说""她说"，也可以省略——灵活度最高 |
| **段落节奏** | 高潮时贴近 → 段落碎、感官密集；过渡时拉远 → 段落长、叙述流畅 |
| **禁忌** | 不能写锚定角色看不到的场景（除非切换聚焦角色），不能随意跳入其他角色的心理（否则是"跳脑"） |

### 2.4 第三人称全知 → 文风影响

| 维度 | 影响 |
|---|---|
| **叙述声音** | 叙述者有独立人格——可以评论、议论、调侃，形成"叙述者的文风" |
| **信息自由** | 可以写任何角色的内心、任何时空的事件 |
| **段落跳跃** | 可以在同一段内从角色A跳到角色B（但需要"重置场景"技巧避免跳脑） |
| **描写风格** | 倾向客观、全景式描写，可以使用角色自己不会用的词汇 |
| **风险** | 现代读者不耐受频繁跳脑——全知≠随意进出所有人脑袋 |
| **最佳实践** | 保持统一叙述声音，进入角色内心前先"重置场景"（重新锚定物理位置） |

### 2.5 第三人称客观 → 文风影响

| 维度 | 影响 |
|---|---|
| **描写风格** | 纯白描——"他放下杯子"而非"他心烦意乱地放下杯子" |
| **对话比重** | 极高——角色心理全靠对话和动作推断 |
| **情感传递** | 冰山法则：水面下的90%由读者自行补全 |
| **段落节奏** | 冷峻、精简、不拖泥带水 |
| **适用场景** | 悬疑/刑侦的案发现场、需要客观距离感的场景 |

## 3. AI 写作中的人称常见失败

| 失败模式 | 具体表现 | 防御规则 |
|---|---|---|
| **视角泄漏** | 第三人称受限中写了锚定角色看不见的事 | 每句检查：锚定角色此刻能感知到这个信息吗？ |
| **跳脑（Head-hopping）** | 同一段落内跳入多个角色的内心 | 进入新角色内心前必须"重置场景"——重新描述物理位置 |
| **人称滑动** | 第一人称写着写着变成第三人称旁白 | 铁律级别禁止人称切换（除非明确设计） |
| **全知降级** | 全知叙述者的声音不统一，像是多个角色的拼凑 | 全知叙述者应有一致的"人格"——用词、语气、评判标准始终统一 |
| **第二人称疲劳** | 长篇第二人称"你做了X，你感到Y"单调重复 | 交替使用行为描写、环境描写、对话打断"你"的连续性 |
| **情感外溢** | 客观视角中混入情绪形容词 | 外聚焦模式下禁止使用情感形容词——只写动作和物理状态 |
| **不可靠叙述者退化** | 设计了不可靠叙述者但全程可靠 | 必须埋入至少 2-3 处"叙述者自己没注意到的矛盾" |

## 4. 人称视角与文风配置的映射规则

在 `<writing_style>` 中配置人称/视角时，应遵循以下规则：

### 4.1 `<narrative>` 区必写项

```
人称与聚焦：[第X人称]，[零聚焦/内聚焦/外聚焦]，叙述距离=[贴肩/适中/远观]
```

这一行是 `<narrative>` 区的**锚定句**——后续所有规则（段落、节奏、对话）都围绕这个锚点展开。

### 4.2 人称决定的级联约束

| 人称×聚焦 | `<narrative>` 影响 | `<expression>` 影响 | `<aesthetics>` 影响 |
|---|---|---|---|
| 第一人称×内聚焦 | 必须定义叙述者声音特征（口头禅、句式习惯） | 描写/对话被叙述者个性染色 | 美学目标 = 叙述者的审美偏好 |
| 第二人称×内聚焦 | 必须定义"你"的代入方式（= 读者 or = 特定角色） | 对话中"你"是否说话、如何标记 | 考虑代入疲劳的节奏控制 |
| 第三人称×内聚焦 | 必须定义叙述距离调控规则（何时贴肩/何时拉远） | 自由间接引语的使用频率 | 美学不受角色认知限制 |
| 第三人称×零聚焦 | 必须定义叙述者人格（是否评论/是否夹叙夹议） | 可以使用角色不会用的高级词汇 | 叙述者美学 ≠ 角色美学 |
| 第三人称×外聚焦 | 必须定义白描边界（禁止写内心） | 对话承载 70%+ 的信息量 | 冰山美学、留白为主 |

### 4.3 AIRP 角色扮演的人称特殊规则

AIRP 场景通常采用「第二人称×内聚焦」（"你"= user 角色）+ 「第三人称×内聚焦」（AI 扮演的角色），形成**双视角交织**：

```
AIRP 人称配置要点：
- "你"的描写：行为+感受，不过度代入读者不想要的情绪
- AI角色的描写：第三人称内聚焦，可以写该角色的内心
- 禁止：AI角色不应写"你"的内心想法——那是 user 的权限
- 对话分离：AI角色对话独立成段，"你"的行为用叙述句承接
```

## 5. 人称组合速查表

供 Phase 2 诊断时快速查阅——根据用户的文风目标，推荐合适的人称配置。

| 用户目标 | 推荐人称组合 | 关键配置点 |
|---|---|---|
| 沉浸式角色扮演 | 第二人称 × 内聚焦 | "你"的代入方式、代入疲劳防控 |
| 意识流/内心独白 | 第一人称 × 内聚焦 | 思维模拟循环、认知标记词 |
| 吐槽役轻小说 | 第一人称 × 内聚焦 | 叙述者声音极强、铁律禁情绪通胀 |
| 传统严肃文学 | 第三人称 × 零聚焦 | 叙述者人格定义、夹叙夹议规则 |
| 现代流行小说 | 第三人称 × 内聚焦 | 叙述距离调控、自由间接引语 |
| 群像/多视角 | 第三人称 × 不定式内聚焦 | 切换规则（章/节为单位）、各角色声音差异 |
| 冷硬派/悬疑 | 第三人称 × 外聚焦 | 白描铁律、对话承载信息量、留白美学 |
| 散文化叙事 | 第三人称 × 零聚焦（偏旁观） | 旁观叙史感、叙述距离适远 |
| AIRP 对话密集 | 第二/三人称混合 | 双视角分离规则、对话排版硬规则 |

## 应用时机

- **Phase 2 诊断时**：根据用户输入判断目标人称组合，参照速查表推荐配置
- **Phase 3 起草时**：`<narrative>` 区的人称锚定句是第一个要写的条目；级联约束决定后续各区写法
- **Phase 4 质检时**：检查视角泄漏、跳脑、人称滑动等 AI 常见失败
