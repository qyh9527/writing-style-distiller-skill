# Phase 1: 研究与证据收集

本阶段负责根据用户输入进行联网深度研究或源文档剖析，收集原文样本和文学分析证据。

## 1. 深度模式：联网搜索规则（作者名输入时）

进行至少 3-5 轮搜索，方向包含：
1. `[作者/作品] 代表作 文风 语言特征 分析`
2. `[作者/作品] 原文 摘录 选段`（必须收集原语种摘录，严禁二传机翻）
3. `[作者/作品] 叙事学 文本批评 论文`（抓取批评学者总结的审美术语）
4. `[作者] 文化美学 传统术语`（如：物哀、陰翳、間等源语言概念）
5. 若作者使用非英语语言，用该语言重复关键搜索。

提取目标：
- 至少 6 段原语种摘录候选。
- 覆盖叙事、表达、美学三大系统的证据。
- 批评术语和文化美学概念。

## 2. 标准模式：文本分析规则（本地文件/摘录输入时）

以用户提供的文本为首要证据来源，不要用外部百科概念覆盖文本本身的表现。

提取指标：
- 对白排版：对白与旁白如何分离？对白是否单独成段？
- 句式特征：流水散句还是密集排比？平均句长倾向？
- 核心情绪与人称视角。
- 段落控制模式：段落长短、情绪转换时的段落变化。
- 感官层级：优先使用哪些感官？

### 2.1 语法层级分析（语言特征类文风必做）

对语言特征类文风，需在标准提取指标之上增加四层分析：

1. **语音层面**：节奏韵律、句式长短交错规律、是否有韵文穿插
2. **词汇层面**：虚词来源（上古/中古/近古/现代）、实词的双音节化程度、是否有特定时代的生活用词
3. **语法层面**：主语省略习惯、定语小句使用频率、流水句 vs 复合句的比例、指示词使用习惯（如"那+名词"是否可省略）
4. **语用层面**：人物对话是否贴合身份（上层典雅 vs 下层俚俗）、情绪激烈处是否反用简笔

### 2.2 反面模式提取

在分析文本正面特征的同时，必须主动识别并记录「该风格下 AI 最容易犯的错误」：

- **该风格的 AI 典型失败模式是什么？**（如古风容易文白混杂、轻小说容易丢失吐槽节奏、NSFW 容易情绪通胀）
- **该风格有哪些"绝对不能做"的事？**（从用户提供的文本或案例中提取铁律，如"严禁情绪通胀""禁止文学替代词"）
- **该风格最容易被哪种八股侵蚀？**（参照 `knowledge/anti-bagu.md`，预判需要重点防御的八股类型）

记录格式：
```
## 反面模式预警
- AI典型失败：[具体描述]
- 铁律候选：[从文本中提取的绝对禁令]
- 八股高危项：[需要重点防御的anti-bagu编号]
```
